
# Pneumonia Detection using Transfer Learning

## Overview
This project investigates the impact of model complexity on pneumonia detection using chest X-ray images and transfer learning.

## Models Used
- MobileNetV2
- ResNet50
- EfficientNetB0

## Dataset
Chest X-Ray Images (Pneumonia) dataset from Kaggle

## Key Findings
- MobileNetV2 achieved the best performance:
  - Accuracy: 89.9%
  - ROC-AUC: 0.972
- More complex models did NOT always perform better
- Model performance was affected by class imbalance

## Tools & Technologies
- Python
- TensorFlow / Keras
- Transfer Learning
- Data Augmentation

## Key Learning
This project shows that lightweight models can outperform complex architectures when working with small or imbalanced datasets.
