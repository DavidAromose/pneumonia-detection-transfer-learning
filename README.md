# pneumonia-detection-transfer-learning
 Conducted a research project exploring the impact of model complexity on AI-based pneumonia detection using chest X-ray images.  Developed and evaluated multiple transfer learning models (MobileNetV2, ResNet50, EfficientNetB0), achieving a top accuracy of 89.9% and ROC-AUC of 0.972. (Python, TensorFlow/Keras, Data Preprocessing, Model Evaluation)
